# 对象导论

##### 抽象过程
~~~
 将问题抽象成机器能够处理的,这是复杂的. 业务与解空间(计算机)对应这是需要丰富的经验.
~~~
面向对象通过向开发人员提供了表示问题空间的元素,表示为 '对象'.
1. 万物皆对象
2. 程序是对象的集合
3. 每个对象都有自己的由其他对象所构成的存储
4. 每个对象都有类型
5. 某一特定类型的所有对象都可以接受同样的消息
